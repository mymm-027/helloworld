QUICK START (recommended)
================================================================================

OPTION 1: Start backend + Shiny (recommended)
1. Backend:
   python -m uvicorn backend.main:app --reload --host 127.0.0.1 --port 8000
2. Shiny:
   Rscript -e "shiny::runApp(appDir='.', host='127.0.0.1', port=3838, launch.browser=TRUE)"

See STARTUP.md for full setup (DB env vars + SMTP).

OPTION 2: RStudio (Shiny only)
1. Open RStudio
2. Open the project: File > Open Project > classroom-emotion-system.Rproj
3. Run: shiny::runApp()
4. Browser opens automatically

OPTION 2: Command Line
1. Open terminal/PowerShell in project directory
2. Run: Rscript run_app.R
3. Open browser to http://localhost:3838 (or shown in console)

OPTION 3: RStudio - Source app.R
1. Open RStudio
2. Open app.R
3. Click "Run App" button (top-right)

================================================================================
TROUBLESHOOTING
================================================================================

App won't start?
  1. Check that all packages are installed (see README.md)
  2. Verify R working directory is correct
  3. Check console for error messages

Charts not rendering?
  1. Make sure you're logged in
  2. Check if data/emotion_records.csv exists
  3. Try deleting CSV and restarting to regenerate

Filters showing "All" only?
  1. Your role might not have access to multiple lectures
  2. Try logging in as admin to see all options
  3. Lecturer role only sees T01's lectures (L1, L3)

Missing packages?
  1. Install packages:
     install.packages(c("shiny", "bslib", "dplyr", "ggplot2", "readr",
                        "tidyr", "lubridate", "DT", "htmltools"))
